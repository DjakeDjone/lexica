# Lern Mode Pro

a seperate page to learn with the help of ai.

## User Experience

1. The user clicks on the "Lern Mode Pro" button next to the "Test" button on the top right of a summary
2. The user is taken to the lexa page but with the tool 'Lern Mode Pro' activated, which provides a more interactive learning experience than the test mode.
3. A start button is displayed, which the user can click to start the learning process.
4. Once the user clicks the start button, an AI creates a learning plan based on the content of the summary and splits the summary into smaller sections.
5. A progress bar is displayed at the top of the page, showing the user's progress through the learning plan.
6. The user is presented with the first section of the summary.
7. The user can read the section and then click a "Memorized" button to indicate that they have memorized the content.
8. After clicking the "Memorized" button, the user is presented with a quiz related to the content of the section they just read.
9. The user answers the quiz questions, and after submitting their answers, they receive feedback on their performance.
10. If the user answers correctly, they can proceed to the next section. If they answer incorrectly, they are given the option to review the section again before moving on.
11. This process continues until the user has gone through all the sections of the summary.
12. Once the user has completed all sections, they receive a final summary of what they have learned and an option to retake the learning process or return to the original summary. The user also can always ask the AI for help or clarification on any section or quiz question.

The AI's focus should always be on the plan it's created. It shouldalways come back and work the sections.

The AI's tools:

- **Edit Plan**: The plan includes a todo list of section to learn which the ai can edit, add to, or remove from. The ai should always come back to the plan and work on the sections in the plan.
- **memorize**: The AI can write to an internal memory to keep track of important things like skills of the user
- ... (more tools to be added)
